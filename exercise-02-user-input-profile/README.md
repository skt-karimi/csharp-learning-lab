# Exercise 02 - User Input Profile

This is a simple C# console application that asks the user for their name, age, and weight.

## What the program does

- Reads the user's name as a string.
- Reads and validates the user's age using `int.TryParse` and a `while` loop.
- Reads and validates the user's weight using `double.TryParse` and a `while` loop.
- Prints a summary message with the user's information.
- Uses an `if/else` statement to check whether the weight is inside a simple normal interval.

## Concepts practiced

- `Console.WriteLine`
- `Console.ReadLine`
- `string`
- `int`
- `double`
- `while` loop
- `TryParse`
- `if / else`
- string interpolation
