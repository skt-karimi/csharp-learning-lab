# Exercise 01 - Variables and User Input

This folder contains my first C# practice exercise about variables, user input, and simple validation.

## Topics

- Creating variables
- Using string, int and double
- Reading input with Console.ReadLine
- Printing output with Console.WriteLine
- Validating input with TryParse
- Using while loops
- Using if and else statements

## What the program does

The program asks the user for their name, age, and weight.  
It checks that the age and weight are valid values.  
Then it prints a short message and checks if the weight is within a simple normal range.
